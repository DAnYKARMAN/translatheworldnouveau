TranslateTheWorld — Mise à jour interface FR/EN
================================================

Trois fichiers à remplacer dans votre dossier Netlify :

1. host-setup.html       → Page de configuration de l'hôte
2. host-session.html     → Interface de l'hôte pendant la session
3. participant-session.html → Interface mobile du participant

Changements :
- Interface en anglais par défaut
- Bouton FR/EN dans chaque écran pour basculer à tout moment
- Limite à 3 langues maximum pour le public (host-setup.html)
- Texte du mode auto clarifié ("Recommandé pour 2 à 3 personnes"
  + avertissement sur la perte de maîtrise des débats)
- Pastille "langues du public" cliquable

Déploiement : glissez-déposez ces 3 fichiers sur Netlify
(ou dans votre dossier local puis redéploiement).
