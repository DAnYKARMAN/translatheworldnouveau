Comment ça marche

Merci d'avoir choisi TranslateTheWorld.

Vous êtes l'hôte de la conférence.
Cela veut dire : vous êtes l'organisateur.

C'est vous qui :
  • 🎬 démarrez la conférence
  • 🎤 parlez en premier
  • 👥 donnez la parole aux participants
  • 🛑 terminez la conférence

1 Vos deux liens

Dans votre email, vous avez deux liens. Ils sont différents.

🟢
Le lien Hôte C'est le vôtre. Gardez-le pour vous.
Vous l'utilisez pour démarrer la conférence.
🟠
Le lien Participants C'est pour vos invités.
Envoyez-le-leur avant la conférence par :
📧 Email 💬 WhatsApp 📱 SMS 📷 QR code

2 Démarrez la conférence

  1. Cliquez sur votre lien Hôte.
  2. Choisissez la langue dans laquelle vous allez parler.
  3. La session démarre.
  4. Partagez le lien avec vos participants par email, SMS, WhatsApp ou QR code en cliquant sur Share en haut de votre interface orateur.
  5. Vous parlez normalement.
  6. Les participants lisent les sous-titres dans leur langue, sur leur téléphone.

3 Donner la parole

Pendant la conférence, un participant peut vouloir parler.

Comment il vous le demande :

  • ✋ Sur son écran, il appuie sur Raise hand
  • 🔔 Sa demande apparaît sur votre écran

Vous décidez :

  • ✅ Pour accorder : appuyez sur Grant. Votre micro se coupe. Le sien s'ouvre.
  • ❌ Pour refuser : appuyez sur Decline. Vous gardez la parole.

Pour reprendre le micro :

  • 🔄 Soit le participant clique Release mic
  • ↩️ Soit vous cliquez Take the floor back

Dans les deux cas :

  • 🎤 Une bannière verte s'affiche : "À vous de parler !"
  • 🔴 Un voyant rouge "MICRO ACTIF" s'allume
  • 💬 Vous pouvez parler
⭐ Règle d'or

Un seul micro est ouvert à la fois.
Soit le vôtre, soit celui d'un participant.
Jamais les deux ensemble.
Cela garantit que les sous-titres restent clairs.
